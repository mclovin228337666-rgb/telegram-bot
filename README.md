# Telegram Schedule Bot v2

Новая версия бота:
- сотрудник вводит ФИО
- бот привязывает его к расписанию
- сотрудник выбирает время напоминания
- бот показывает его смены и напоминает перед каждой сменой

## Что менять каждый месяц
Меняй только файл `app/schedule_seed.py`.

## Команды и кнопки
- /start
- Сегодня
- Завтра
- Моё расписание
- Напоминание
- Сменить ФИО
